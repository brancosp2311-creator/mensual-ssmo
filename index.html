Dashboard de ingresos de accidentes — conectado a Google Drive vía n8n

Cada vez que actualices los archivos en Drive, al recargar verás los datos frescos.

⚠ Estás abriendo el archivo desde file://. Algunos navegadores bloquean solicitudes externas desde archivos locales.
Sube el archivo a GitHub Pages o usa un servidor local para que el fetch funcione correctamente.

Primeros pasos

1
Asegúrate que el workflow API Datos SSMO esté activo en n8n
2
Selecciona el mes y año del período a visualizar
3
Haz clic en ⟳ Cargar datos — toma ~15–20 segundos mientras n8n descarga desde Drive
4
Para GitHub Pages: sube este archivo como index.html en un repositorio público
